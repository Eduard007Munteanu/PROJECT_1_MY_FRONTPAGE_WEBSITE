Deep Learning for Visual Recognition: YOLO-Style Object Detector From Scratch

Project Description

This project studies a simplified YOLO-inspired grid classifier for football image analysis. The model uses a compact convolutional backbone and a classification head that predicts whether each grid cell contains a player or the ball. Rather than relying on a large pre-trained object detector, the project focuses on understanding how much performance can be gained from a small model through better training strategies.

A central challenge was the strong imbalance between the two classes. Player cells appeared far more often than ball cells, meaning that the baseline model could achieve high overall accuracy while still failing to recognise the football reliably. To investigate this, we compared several approaches, including class-weighted loss, random oversampling, hybrid sampling, cyclic learning rates, and hard negative mining.

The final pipeline showed that the most effective improvements came from combining hybrid sampling with cyclic learning and hard negative mining. This made the model focus more on difficult ball examples and improved minority-class performance without sacrificing player accuracy.
