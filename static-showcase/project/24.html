Advanced Augmented Reality Project : AR Kingdoms

AR Kingdoms is an AR/VR tabletop strategy prototype for Meta Quest 2, where the player rules a virtual kingdom placed on a real table using hand tracking, gaze/pinch selection, direct touch interaction, and card-based building placement.

Project Description

This project explored how a physical table and surrounding room can be used as part of a mixed-reality strategy interface. The prototype used spatial anchors, hand tracking, gaze/pinch selection, direct touch interaction, and card-based placement to connect virtual game objects with real-world surfaces.

The design process moved through several stages: early ideation from cultural themes, sketching and concept selection, mockups, think-aloud playtesting, and a final EXPO evaluation. Each phase helped refine the interaction flow, especially how users selected buildings, interacted with citizens, and understood the system state.

A central focus was environmental feedback. Instead of showing everything through standard menus, the prototype used candles, paintings, walls, and data panels to communicate progress, happiness, resources, and political state. The EXPO feedback showed that the atmosphere and tabletop concept were effective, while also highlighting areas for improvement such as tutorial clarity, interaction mapping, and clearer communication of the project’s underlying themes.
