Bachelor Thesis - BlobbyKelp Data Visualization

BlobbyKelp is a bachelor thesis prototype for organic data visualization, representing grouped data points on maps and 2D graphs through soft, blobby shapes. The project focuses on transforming rigid geometric groupings into more readable and visually natural shapes, making clusters easier to interpret.

Project Description

This bachelor thesis explores how grouped data points can be visualized more organically while still staying readable and structurally meaningful.

The prototype, called BlobbyKelp, was built to move beyond rigid geometric clusters and create a softer visual language for maps and 2D graphs. It draws inspiration from GapMinder, Kelp Diagrams, BubbleSets, and KelpFusion, but aims for a more continuous and organic result.

To achieve that, the project combines graph routing, local grouping, hull generation, and curve smoothing. Related points are connected efficiently, grouped regions are shaped through convex and non-convex hull logic, and spline-based smoothing helps transform sharp boundaries into blobby forms that feel more natural to read.

A large part of the work was about balancing visual clarity with technical control. The final prototype supports animation and adjustable thresholds, making it easier to inspect how grouped data evolves and how different clustering choices affect the output.
