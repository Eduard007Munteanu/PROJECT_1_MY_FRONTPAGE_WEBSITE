Academic Project

Master Thesis - GRIP-TK Toolkit

GRIP-TK is a master thesis toolkit for creating and evaluating XR interaction techniques through a node-graph workflow. It brings interaction logic, runtime behavior, logging, and evaluation setup into one system, reducing the amount of repeated work normally needed when building XR experiments.

Academic Project

Bachelor Thesis - BlobbyKelp Data Visualization

BlobbyKelp is a bachelor thesis prototype for organic data visualization, representing grouped data points on maps and 2D graphs through soft, blobby shapes. The project focuses on transforming rigid geometric groupings into more readable and visually natural shapes, making clusters easier to interpret.

Academic Project

Building the Internet of Things with P2P and Cloud Computing : AirBrush Project

Airbrush is a multiplayer browser drawing game where players use phone motion sensors to draw in the air, while others guess the drawing on a shared canvas. The project explores WebRTC-based peer-to-peer communication and IoT-style motion interaction.

Academic Project

Advanced Augmented Reality Project : AR Kingdoms

AR Kingdoms is an AR/VR tabletop strategy prototype for Meta Quest 2, where the player rules a virtual kingdom placed on a real table using hand tracking, gaze/pinch selection, direct touch interaction, and card-based building placement.
